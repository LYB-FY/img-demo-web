# 图片相似度校验工具

基于 Next.js 和 MobileNetV2 的图片相似度比较工具。

## 功能特性

- 🖼️ 支持拖拽上传图片
- 📊 使用 MobileNetV2 深度学习模型提取图片特征
- 🔍 计算两张图片的余弦相似度
- 📱 响应式设计，支持移动端

## 技术栈

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **TensorFlow.js** - 机器学习库
- **MobileNetV2** - 轻量级图像分类模型

## 安装

```bash
npm install
```

## 运行

开发模式：

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

## 构建

```bash
npm run build
npm start
```

## 使用说明

1. 上传第一张图片（支持点击或拖拽）
2. 上传第二张图片
3. 点击"比较相似度"按钮
4. 查看相似度得分和评价

相似度得分范围：0-100%
- 90%+ : 非常相似
- 70-90% : 相似
- 50-70% : 中等相似
- 30-50% : 不太相似
- 0-30% : 不相似

## 工作原理

1. 使用 MobileNetV2 模型提取每张图片的特征向量
2. 计算两个特征向量的余弦相似度
3. 将相似度转换为 0-1 范围的分数
